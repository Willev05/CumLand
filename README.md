# Syntax

Output:
- Cum on var
Input:
- Cum in var

In for input and on is for output!

Defining variables:
- var is value

Addition:
- input_var plus input_var is output_var
Subtraction:
- input_var minus input_var is output_var
Multiplication:
- input_var times input_var is output_var
Division:
- input_var devided input_var is output_var
